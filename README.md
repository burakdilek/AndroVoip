# AndroVoip
This is the Simple Softphone aplication based on android-ngn-stack library.

**Project Owners:**
Hamdi Burak Dilek <br>

**Features:**

- **Calling (Sip server)**
	- Speaker(On/Off)
	- Mic(Mute/Not)
- **Message (XMPP Server)**
<br>
-  **Contacts (Local SQLite)**
	- Message Shortcut
	- Call Shortcut
	- View Profile
	- Add Contact
	- Update Contact
	- Delete Contact
